ARP Spoofing Alarm

This is a simple security tool built with Python and the Scapy library.

It monitors ARP activity and, if it detects a change in the router's IP address, it generates an alarm.
Features: A good tool for home security.
Disadvantages: Not suitable for large networks.
Requirements: pip install scapy

Run: sudo python3 Firewall_ARP.py

Note: Check your interface name in the code.
